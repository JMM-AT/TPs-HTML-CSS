# Projets HTML/CSS - TPs et Démos

Bienvenue dans ce dépôt de travaux pratiques (TP) et exercices HTML/CSS. Il présente une progression pédagogique de la structure de base HTML/CSS aux interfaces multi-pages responsives et visuelles modernes.

---

##  Arborescence du dépôt

- **`/EX1`...`/EX6`** : Exercices individuels sur HTML et CSS.
- **`/tp1`** : Premier TP (structure HTML basique, mise en forme simple).
- **`/tp2`** : Deuxième TP (layout CSS, formulaires, utilisation de Flexbox).
- **`/tp3`** : Troisième TP (site multi-pages complet, design moderne, responsive, composants avancés).

---

##  Aperçu des TP

### `tp1`
- Page d'accueil simple, balisage HTML de base, styles CSS locaux.

### `tp2`
- Mise en page responsive, formulaires, gestion des grilles et du positionnement.

### `tp3` (Agences digitale)
- Site vitrine multi-pages (accueil, à propos, services, galerie, contact)
- Navigation interne et contenu structuré
- Modes sombre/clair et animations de survol
- Effets *Glassmorphism*, grille CSS avancée, typographie soignée

Pages présentes:
- `index.html`
- `pages/about.html`
- `pages/service.html`
- `pages/gallery.html`
- `pages/contact.html`

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3 (Flexbox, Grid, variables CSS, animations, transitions)
- Sans frameworks externes (tout en code natif)

---

## ▶ Exécution

1. Ouvrir le fichier HTML dans un navigateur (double-clic ou via la commande `Live Server`).
2. Pour une meilleure expérience, utiliser un serveur local (ex: Visual Studio Code `Live Server`).
